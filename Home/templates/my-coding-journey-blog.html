hi guys 👋 welcome to my blog. well, this is my first blog, its all about how i got into coding and making web apps and loving it!


the start of the coding journey!

so the story starts from ‘bout 6 years ago, i was in 7th standard, studying in podar international school, nashik, maharashtra. in school we(we = all my classmates, bro) were taught html as our first language, were taught to make web pages, make tables, insert images and other basic stuff and since then my hunger for learning web languages and making web pages increased, i became passionate about making web pages, yeah i know i was a noob at that time and i still am but you learn on the way. would code html only in school during the computer lab time. i knew just plain html at that time, no css no nothing.

Then got into 8th standard and were still being taught html, but it was kinda advanced html, consisted of input tags and buttons and stuff. There was no flexbox taught at that time which is 6 years ago, and now you can't make a single page without using flexbox😝 .


some side projects

The one thing i did in 8th standard that i’m very proud of is that i learned 3d printing and 3d designing, our teacher taught 3d designing in our school computer lab using tinkercad, which was the only software which we knew could help us design stuff in 3d. i had made crazy designs for inter-school competition, and guess what, we won the competition, i donno if we won because of me but you know.....😌 😉 hehe. here’s what i made at that time based on the musical instruments theme of the competition. these 3d designs seemed soo crazy to 13-year-old ankoor🤯 and they still are!




entry of a real programming language

transition to class 9th... now 9th standard was one of my best school years! just because all of my best friends were in the same class. i was much more interested in programming than my friends, so i used to everyone that i'm a hacker because i know html, but that 'i'm a hacker' greatness was for a short time until the real gangster entered the chat, java. yes java, only java not script😉. we were taught java in 9th standard and honestly, java was a headache for me at that time.

the syllabus was full of object-oriented programming we were taught to define functions and call classes and constructors and destructors, that's all i remember. then passed 9th standard with not so good % cause you know, you can't study when all your besties are around, it's just enjoyment. then comes the 10th standard, in 10th had only some of my best friends in my class cause management used to shuffle the children into different classes each year. now talking about programming, in 10th too we were taught java, but now all that find prime number, and palindrome and create arrays and advanced stuff. i started to love programming in java because.... i donno why i liked java!🙃 but it was like i knew how to program and run the program without any errors and then used to roam in the computer lab solving doubts of the girls... and yes, doubts of boys too🤭😁. Then passed 10th with 98 marks in computer studies, sick right! didn’t give the cs exam cause the coronavirus pulled up! but gave 6 exams and based on that got marks for the cs exam. i had 94% in 10th. i don’t believe it. 😂


learning python and django

as the pandemic started, i used to watch tv and play nfs underground 2, all day! then i started watching a program on history tv 18 called shipping wars. i liked that show a lot because it had that competition kinda vibe in it. some bunch of people used to bid on stuff to ship it from one place to another and watching that program, an idea of making a web app showed on the program struck my mind. i didn't knew anything at that time, didn't knew what product market fit, marketing, business plans, i just wanted to do something great and buy a lambo😉.

i then started researching on how you make web apps and what languages used to make those kinda web apps simultaneously while preparing for jee(i’m kidding, i wasn’t and i’m still not studying😅😭). after a lot of research went ahead to learn python, ordered a book called python crash course , by eric matthes and then started learning from YouTube, yeah!🙂 i watched a youtube video by mosh hamedani, bruh, he's the best coding teacher out there. at the end of the video, mosh introduced me to a python framework called django and i was amazed by what all stuff django could do and so easily.

Then i started making the web app. it took me months because that was the first time i was using django, it was my first project and i didn’t even learn the django framework, just started and that’s the thing i admire about myself. i just start doing things and learn on the way! self love bruh! yeah i left that project in the middle cause i couldn’t handle that entrepreneur vibes(jk)😉, no but i left building it any further cause dad said that you should stop doing these things and focus on studies. so i did what he said.


gettin' back to my passion

after leaving the project in the middle, i started studying, started the preparation for jee which was just 8 months away. when ever i used to open my laptop to get going with the zoom classes, i used to stare at pycharm, my code editor(a software bro, not a girl in my class😏). i resisted. i was far away from my ❤️. i deleted pycharm, github account and everything(deleting the github account was the worst decision of my life😭). I studied till my board exams, which are now over. but the distance didn't last long(it was long enough for me tho🥹), just after two months on my birthday, i got a new laptop, so i just wanted to see how pycharm works in mah new lapi. i opened pycharm

and then, there was no deleting it,

that means, i was back, but as a result of taking a big break from coding i forgot how to even start a new django project! forgeting the basics was a nightmare but then it was going kinda great, i'm was recalling stuff. i learned it all by myself so recalling things was easier.


birth of hyperlynk

as a result of a big break, my self-confidence level was on the ground, there was only one way that i could regain my self-confidence and recall all development processes, which was by starting a new project, so i started making a link in bio tool, yes and that’s how hyperlynk was born. hyperlynk is the free link-in-bio tool that pays users, yes it pays users! but it isn’t launched yet. you gotta have patience, hyperlynk could be your side hustle. just make a hyperlynk and get money in your bank account at the end of every month😉.

while learning django i also learned tailwind css, it's dope, easy to work with and powerful library, made the new u.i. of hyperlynk with tailwind. loved tailwind but i still like vanilla css(i like crying😅😭), no cap. i love animating stuff using keyframes rule, my favourite part of using vanilla css is that when you use linear or radial gradient and it actually looks cool, feels like heaven!😌


break pt. 2

everything was going great, was coding for atleast 5 hours with studying but then again dad said that you again gotta stop coding bro, need to study for jee exams, gotta get into iit bombay(the dream of every parent in india), so i stopped coding for like 4 months. i was missing to code, was missing the journey of buidling stuff, was missing the anger and worthless feeling that comes when you miss a semicolon and cry all night, was missing the, 'this thing could work' thoughts which would come while brushing my teeth.


comeback pt. 2

so, after a long break of 4 months, as soon as my 12 board exams were done, i started working on my portfolio, this is what i’ve created after finishing my exams.

That’s how my coding journey is! a lot of breaks, a lot of sleepless nights, a lot of why tf this ain’t working, a lot of this a lot of that, hope this journey is gonna continue, this blog will get updated and i hope you liked this blog, hope you learned something.

thank you guys,
